<h2 id="1">CRC</h2>
<p>
    - Cyclic Redundancy Check <br>
    - Polynomial Code <br>
    - Message converted to a binary number <br>
    - Both sender and receiver divide the message by gp. <br>
    - Remainder is added to the sent message. <br>
    - Thus, 0 remainder means no error. <br>
</p>
